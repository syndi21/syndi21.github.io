关于网站导航

导航栏是网站最重要的设计元素：

  • 引导用户访问首页之后的页面
  • 给用户方位感的奇妙工具

一个网站的不同页面使用同一导航条，方便不同页面之间的跳转之外，也是网站风格统一的标识。

网站导航一览

网站导航的设计

  1. 确定网站所有内容，正确分析和组织成符合逻辑的、高效的结构。
  2. 导航栏中的项目不宜过多
    • 导航栏中的链接项数5~8为最佳，也正好适合网页宽度；
    • 太多的链接选择会影响浏览者判断，可读性变差，可能会超出页面的宽度。
  3. 项目用语简洁
    • 导航栏中使用精确明了的词语。避免过长。
    • 尽量使用约定俗成的词语，比如“联系我们”，而不用“取得联系”。
  4. 正确排布
    • 从左到右的阅读习惯，依栏目的重要性或内在逻辑顺序从左到右的排布；
    • 导航栏带见安排在网页横幅上，也可在页面侧边安排竖式的导航条。

}

color: #FFF;
}

a：关于超链接字体颜色等基本的设置

CSS提供了四种a对象的伪类：

  • a:link { }——未访问过的链接样式
  • a:visited { }——链接访问过后的样式
  • a:hover { }——鼠标经过时的超链接样式
  • a:active { }——鼠标点击时的链接样式

下划线说明：

  • text-decoration: none; 无下划线
  • text-decoration: underline; 带下划线

注意顺序：“爱恨原则”（LoVe/HAte），即四种伪类的首字母:LVHA。

图像翻转链接

设计步骤

  1. 准备两个同尺寸的翻转图像；
  2. 在a标签选择器中设置：
    • 链接文字基本属性
    • 显示为block方式
    • block的大小（参考图像大小）
    • 背景图像设置
  3. a:link设置未访问过的链接样式；
  4. a:hover设置鼠标经过时的链接样式：
    • 有无下划线
    • 文字尺寸与颜色
    • 背景图像设置

设计步骤：

  1. 项目列表css样式.bar设置（项目符号等）；
  2. 给每项设置超级链接；
  3. 可用.bar li设置项目样式（块显示、块尺寸或内间距、背景色等）；
  4. .bar li a进一步设置链接样式（块显示、链接字体尺寸与对齐方式等）；
  5. .bar li a:hover设置鼠标经过时的链接样式（背景色、字体）。
