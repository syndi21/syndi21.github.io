JavaScript认识

  • JavaScript 是一种脚本语言，用以编写客户端程序；
  • JavaScript 程序嵌在HTML中使用，解释性，不需预编译；
  • JavaScript 基于对象，由事件驱动；
  • JavaScript主要功能有：
    • 验证表单、用户交互操作。如网上问卷等
    • 检测浏览器，控制浏览器行为。如浏览器的退后、刷新、加入收藏夹等。
    • 与页面元素交互，如控制图片等。
    • 读写客户端信息，即客户端的cookies信息。

 

 

网页中使用JavaScript

JavaScript脚本格式

<script type="text/javascript">
JavaScript代码行
</script>


<html>
<body>
<script typle="text/javascript">
var i,j;
for(i=1;i<10;i++){
document.write("&nbsp;&nbsp;&nbsp;&nbsp;");
for(j=1;j<=i; j++){
document.write("*");
}
document.write("<br/>");
}
</script>
</body>
</html>
  • document.write 字段是标准的 JavaScript 命令，用来向页面写入。大小写敏感。
  • 如输出多行，行之间的换行不可用”\n”，需用HTML中的<br>或别的标签实现。
  • 空格用&nbsp;来输出。
  • 语句后分号可选，可用分号一行写多条代码。

脚本的位置

脚本一般放在 head 部分（推荐）， 也可放在 body部分。

<html>
<head>
<script type="text/javascript">
....
</script>
</head>

<body>
<script type="text/javascript">
....
</script>
</body>
</html>

脚本可单独写在外部文件(.js)中
外部文件的使用：

<script type="text/javascript" src="xxx.js">....</script>

信息确认框

confirm("文本")

确认框用来让用户进行信息验证或接受信息。 返回值：

  • 用户点击确认，返回值为 true。
  • 用户点击取消，返回值为 false。

例：将显示三角形的JS程序嵌在一个带id（设为star）的div中，在显示三角形前作颜色选择，并直接更改此div的color样式值。

<div id="star">
<script type="text/javascript" src="star.js">
</script>
</div>
if(confirm("Would you like to show in red?")){
document.getElementById("star").style.color="red";
}

信息输入框

prompt("文本","默认值")

提示框常用于提示用户在进入页面前输入某个值。 返回值：

  • 返回值为输入的值
  • 如用户点击取消，那么返回值为 null

例：在三角形输出前再加上像素尺寸的输入，以确定三角形中星号的大小。

document.getElementById("star").style.fontSize=prompt("Please input the font size what you wanna show:")+"px";

JavaScript 变量名称的规则

  • 变量对大小写敏感（y 和 Y 是两个不同的变量）
  • 变量必须以字母或下划线开始

常用事件

事件名 事件发生时机
onabort 图像加载被中断
onblur 元素失去焦点
onchange 用户改变域的内容
onclick 鼠标点击某个对象
ondblclick 鼠标双击某个对象
onerror 当加载文档或图像时发生某个错误
onfocus 元素获得焦点
onkeydown 某个键盘的键被按下
onkeypress 某个键盘的键被按下或按住
onkeyup 某个键盘的键被松开
onload 某个页面或图像被完成加载
onmousedown 某个鼠标按键被按下
onmousemove 鼠标被移动
onmouseout 鼠标从某元素移开
onmouseover 鼠标被移到某元素之上
onmouseup 某个鼠标按键被松开
onreset 重置按钮被点击
onresize 窗口或框架被调整尺寸
onselect 文本被选定
onsubmit 提交按钮被点击
onunload 用户退出页面
