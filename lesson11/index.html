表单与表单设计

网页上的表单

网页表单是验明用户身份、收集用户信息的最简单和直接的办法。

表单设计原则

良好设计的表单可提供有效信息, 网页表单设计应遵循一定的规则：

1、简洁

  • 保持表单结构简洁，表单内的每个元素是有意义的；
  • 考虑人们的阅读习惯：从上到下, 从左到右；
  • 可Tab键在表单的各个元素之间移动；
  • 标签作为提示文字表达准确，无二义性；
  • 所有的表单元素均匀的分隔, 排列整齐。

三种常见的标准表单：

  • 标签左对齐的表单显得更整齐有序
  • 标签右对齐的表单相对容易区分和信息录入
  • 标签顶端对齐的表单整齐感不如前两种，但可令用户更便利填写表单

2、以用户为中心

表单设计应以用户为中心，以达到良好的沟通，避免呆板的设计。

  • 需要用户输入的信息应是简明的
  • 对于答案为固定项目的信息，可通过下拉列表和单选按钮让用户选取的方式作答
  • 对敏感的个人信息声明隐私保护
  • 较多信息量的文本框下给出简明扼要的提示
  • 输入信息错误的提示

3、表单区域划分

区域的划分有利于用户对表单的阅读。

区域的划分方法：

  • 用水平线分隔不同的区域
  • 不同的色块
  • 用标题文字分割信息

 

表单的分类

搜索/查询表单、注册/登录表单、意见薄表单、订单输入表单、邮件编辑发送表单、 ……

 

 

表单常用控件

form标签

 <form>是表单的标记，可以看做一个包含很多表单控件的容器，为块状元素。基本语法结构是：

<form> 表单的内容 </form>

<form action="form_action.asp" method="post">
<label for="username">姓名</label>
<input type="text" name="username">
<input type="submit" value="提交">
</form>

form的主要属性

  • action：规定提交表单时向哪个文件发送表单数据。如：action="form_action.asp"、action="form_action.jsp"；
  • method：用来设定将表单数据传输到服务器所使用的方法，可取的属性值有：
    • get:将表单数据附加到所请求的RUL中，此种方法不能传送大量数据，安全性差；
    • post:将表单数据嵌入HTTP请求中，此种方法容许传送大量数据，较为常用。

加入表单到页面

制作步骤：

  • 插入表单
  • 在表单中布局
  • 插入表单元素

 

input标签

根据不同的 type 属性值，表现为不同的控件：文本框、复选框、密码输入框（掩码后的文本控件）、单选按钮、按钮等等。

type属性 说明
text 单行文本输入框
checkbox 复选框
radio 单选按钮
password 密码输入框(输入的文字用*表示)
submit 提交表单信息，提交给action属性所指向的文件
image 图像按钮，提交表单信息
  • 通过文本框可输入字母、数字等内容 ，文本框的默认宽度是20个字符
  • 一个表单中的多个单选按钮，互相排斥
  • 复选框适用于给出多项，供用户选取其中一个或若干选项

select标签

<select name="fruit" id="fruit">
<option>Apple</option>
<option>Peach</option>
<option>Papaya</option>
<option>banana</option>
<option>Orange</option>
</select>


textArea标签

<textarea name="yoursuggest" cols ="40" rows = "5">
</textarea>

通过 cols 和 rows 属性规定 textarea 的尺寸，使用 CSS 的 height 和 width 属性更精确。

提示与说明：

  1. 注册表单为335宽，350高（单位为像素）；
  2. 网页中有效图文部分应置页面的中心；
  3. 定位好div盒子（自行设置样式reg），在div中放入表单：

<div class="reg">

  1. 表单上属于一组的标签与文本框，可放在同一个div中：

<div class="textfield">
<label for="user">USERNAME</label>
<input type="text" name="user" id="user" />
</div>

  1. 标签与文本框及其放置的div的样式为：

.textfield {
position:relative;
margin:5px auto;
padding:7px;
height:40px;
width:272px;
text-align:center;
}
.textfield input {
display:block;
height:40px;
width:272px;
text-align:center;
font-size:18px;
}

 .textfield label {
position:absolute;
display:block;
height:40px;
width:272px;
line-height:40px;
vertical-align:middle;
color:#999;
}
  1. 图片按钮需在input标签中，将类型type设为image，同时设置src属性，指定图片文件；
  2. 点击文本框，准备输入时，提示标签自动清空，对应的Javascript代码为：

<script type="text/javascript" src="jquery.js"></script>
<script type="text/javascript">
$(document).ready(function() {
$('.textfield label').click(function() {
$(this).hide();
$(this).parent().find('input').attr('value', '').focus();
});

$('.textfield input').focus(function() {
$(this).parent().find('label').hide();
if( this.value != '' ) {$(this).select();}
});

$('.textfield input').blur(function() {
if( this.value == '' ) {
this.value = '';
$(this).parent().find('label').show();
}
});
});
</script>
